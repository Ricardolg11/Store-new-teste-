## 📝 Resumo do Projeto

O **Store da Nail** é uma solução de software web projetada para centralizar e profissionalizar o gerenciamento de inventário de uma microempresa de cosméticos que opera em modelo home-office. O sistema resolve a fragmentação de dados oriundos de canais informais (WhatsApp e Instagram), permitindo o controle rigoroso de datas de validade por lote, organização por marcas preferenciais e automação do registro de entradas e saídas de produtos.
